# 1 文件路径

一般来说：

* `./`​表示的是当前目录

* `../`​表示的是父级目录

## 1.1 PowerShell

文件路径示例：

```powershell
C:\Users\Default\Desktop
```

使用的是***反斜杠*** `\`​

## 1.2 Git Bash

文件路径示例：

```bash
/d/0Program/00OnlineTech/2025WinterVacation/iForest
```

使用的是***斜杠*** `/`​

## 1.3 Python

文件路径示例：

```python
D:/0Program/00OnlineTech/2025WinterVacation/iForest
```
